# AlmondiNimes рҹҢҷ

AlmondiNimes es una aplicaciГіn Android moderna diseГұada para que los entusiastas del anime y manga puedan gestionar sus listas, descubrir novedades y conectar con amigos. El proyecto estГЎ enfocado en el rendimiento, la escalabilidad y una experiencia de usuario fluida con un diseГұo oscuro elegante.

## рҹҡҖ TecnologГӯas y Arquitectura

- **Arquitectura:** MVVM (Model-View-ViewModel).
- **Concurrencia:** Kotlin Coroutines para operaciones asГӯncronas y de red.
- **Red:** Retrofit con cachГ© de 10MB e interceptor para soporte offline.
- **UI:** 
    - `DiffUtil` en todos los adaptadores para actualizaciones eficientes.
    - Fragmentos unificados para reducir redundancia.
    - Scroll infinito centralizado.
- **NavegaciГіn:** Jetpack Navigation Component con gestiГіn centralizada en `NavegacionUtil`.
- **API:** IntegraciГіn con Jikan API (MyAnimeList).
